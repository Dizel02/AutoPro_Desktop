# AutoPro — Desktop-приложение для автосервиса

Интуитивно понятное десктопное приложение для автоматизации работы автосервиса. Программа позволяет управлять базой данных услуг, мастеров и заказов, поддерживает разграничение прав доступа (Администратор/Пользователь), ведет строгий аудит действий и предоставляет аналитическую статистику в реальном времени.

## 🛠 Стек технологий
* **Язык программирования:** Python 3.13
* **Графический интерфейс (GUI):** Tkinter (встроенная библиотека Python)
* **База данных:** SQLite 3 (встраиваемая реляционная БД)
* **Формат экспорта:** CSV

## 📥 Как установить (по шагам)

Для работы приложения не требуется установка тяжелых сторонних серверов баз данных. Всё необходимое уже встроены в язык Python.

1. **Установите Python:**
   * Скачайте Python версии 3.11 или выше с официального сайта [python.org](https://python.org).
   * **ВАЖНО:** При установке на Windows обязательно поставьте галочку напротив пункта **"Add python.exe to PATH"** в самом первом окне установщика.
2. **Скачайте проект:**
   * Склонируйте данный репозиторий кнопкой `Code -> Download ZIP` или выполните команду в терминале:
     ```bash
     git clone https://github.com
     ```
3. **Распакуйте архив** (если скачивали ZIP) в удобную для вас рабочую директорию (например, `D:\AutoPro`).

## 🚀 Как запустить (по шагам)

1. Откройте редактор **VS Code** (или стандартную командную строку Windows).
2. Выберите пункт меню `File -> Open Folder` и откройте папку с проектом (`D:\AutoPro`).
3. Откройте встроенный терминал с помощью комбинации клавиш `Ctrl + ~` (или `View -> Terminal`).
4. Убедитесь, что терминал находится в папке проекта, и введите команду запуска:
   ```bash
   python app.py
   ```
5. На экране появится графическое окно авторизации.
